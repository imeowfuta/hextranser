this tool coverts the colors from any plaintext theme file into an image where each pixel represents a color (left to right, row by row)
you can then download and edit that image in a graphics program and reupload it to insert the changed colors back into the theme file
it supports hex and rgb, as well as alpha channels


for example: color select and hueshift all the corpo blues #50A0F0 to gentle faded pinks #ff69aaaa
or paint over the generic mass market grays (120,120,120,220) with a fiery people's red (210, 1 ,2)
